# Display Toggle

A simple Windows utility to quickly switch between single and extended display modes.

## Requirements

- Windows 10/11
- Python 3.x
- [customtkinter](https://github.com/TomSchimansky/CustomTkinter)

## Installation

Install the required dependency:

```
pip install customtkinter
```

## Usage

Run the script:

```
python toggle_display.pyw
```

A small GUI window will appear showing your current display mode (**Extended** or **Single**). Click the button to toggle between modes.
